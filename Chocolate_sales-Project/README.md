# 📊 Chocolate Sales Dashboard (Excel)

## Project Overview

This project focuses on analyzing chocolate sales data using Microsoft Excel to uncover business insights and performance trends. The dashboard was designed to provide a high-level overview of key sales metrics, product performance, geographic performance, and salesperson effectiveness.

Using Pivot Tables, Pivot Charts, formulas, and dashboard design principles, the project transforms raw sales data into a clear and actionable business report.

---

## Objectives

The primary objectives of this project were to:

* Analyze overall sales performance
* Track total boxes shipped
* Identify top-performing products
* Determine the highest-performing countries
* Evaluate salesperson performance
* Build a professional sales dashboard for business reporting
* Practice Excel-based data analysis and visualization techniques

---

## Dataset Information

The dataset contains transactional chocolate sales data, including:

* Order dates
* Product names
* Sales amounts
* Boxes shipped
* Salespersons
* Countries
* Customer orders

This information was used to generate key performance indicators (KPIs) and business insights.

---

## Key Performance Indicators (KPIs)

The dashboard highlights several important business metrics:

### Total Sales

Measures the total revenue generated across all transactions.

### Total Boxes Shipped

Tracks the overall shipment volume.

### Top Country

Identifies the country contributing the highest sales revenue.

### Top Product

Determines the product generating the most revenue.

### Top Salesperson

Highlights the highest-performing salesperson based on sales generated.

## Dashboard Limitation

This dashboard was developed using the free version of Microsoft Excel available to me at the time of the project.

As a result:

* Interactive features such as slicers and advanced dashboard controls were not available.
* The dashboard is presented as a static reporting dashboard rather than a fully interactive business intelligence solution.

Despite this limitation, all calculations, visualizations, KPI tracking, and analytical components were built manually to replicate the functionality of a professional sales reporting dashboard.
